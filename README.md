# NHTSA-Complaints-Report
In this project, I modeled a public dataset from the National Highway Traffic Safety Administration using Power Query to shape the data and build relationships, and DAX to create custom measures. The result is a Power BI report that analyzes trends from over 92,000 vehicle complaints across North America.
